* NVIDIA RTX GPUs (RTX 3060ti and RTX 5070ti) running consistantly at 100% heats up the GPU temperature to 85C
For a Nvidia GPU or if youâ€™ve got multiple NVIDIA GPUs in your system, you can use nvidia-smi to set a separate power limit per GPU.

Step1: Get your GPU info
$nvidia-smi -L
Output should be something like this

GPU 1: NVIDIA GeForce RTX 3060 Ti (UUID: GPU-b0a2f09b-feca-6bc3-47e5-79f0d8494baf)

Step 2: Get allowed power limits configurations for each GPU
$nvidia-smi -i 0 -q -d POWER
$nvidia-smi -i 1 -q -d POWER

Step 3: Now set your desired custom limits \
It must has to be in between the Min power and Max power limit from configuration above\
$sudo nvidia-smi -i 0 -pl 280
$sudo nvidia-smi -i 1 -pl 190

Step 4: Verify the new power configurations
$nvidia-smi --query-gpu=index,power.limit --format=csv
Output:
index, power.limit [W]
0, 280.00 W
1, 190.00 W

* Permanent settings: This settings only work temporarity until you reboot. To make it permanent you need to script it in the /etc/
First, create a systemd service 
$sudo nano /etc/systemd/system/nvidia-powerlimit.service
Then, add the following [give your own desired wattage for the GPU]
[Unit]
Description=Set NVIDIA GPU Power Limits
After=default.target

[Service]
Type=oneshot
ExecStart=/usr/bin/nvidia-smi -i 0 -pl 280
ExecStart=/usr/bin/nvidia-smi -i 1 -pl 180
RemainAfterExit=yes

[Install]
WantedBy=multi-user.target

* Save and exit the systemd service
* Reload and enable service
$sudo systemctl daemon-reload
$sudo systemctl enable nvidia-powerlimit.service
$sudo systemctl start nvidia-powerlimit.service

* Verifty that system service works after reboot & runs stably
$nvidia-smi --query-gpu=index,power.limit --format=csv
